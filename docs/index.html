Verkefni 3 Rasberry-PI

Verkefnið gengur út á það að búa til LED Matrix Panel með Rasberry PI. Á skjánum á að birtast litlir punktar sem að hreyfast eftir því hvernig þú snýrð skjánum. Vinnuferlið er mestmegnis verklegt en það þarf líka að setja upp alls konar drivera og kóða inn á Rasberry PI tölvuna áður en maður fer að setja allt saman. Svo þarfnast basic kunnáttu á að lóða saman víra við hvern annan og við board.

Við fylgdum eftirfarandi tutorial við að setja saman og setja upp Rasberry PI tölvuna: https://learn.adafruit.com/matrix-led-sand/overview

GitHub repository sem inniheldur nauðsinlega kóðann: https://github.com/adafruit/Adafruit_PixelDust
